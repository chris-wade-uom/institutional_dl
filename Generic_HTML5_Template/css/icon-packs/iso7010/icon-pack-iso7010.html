Icon Pack: ISO 7010 Safety Signs

The ISO 7010 is an international standard for consistent safety sign regulation across Europe. This icon pack collects together all the symbols that form part of this standard into the five different symbol groups.

These styles can be used with boxes to present messages to users as illustrated here. The CSS styles have been organised such that you can either add all icon symbols to your project/page or just a sub-collection:

Collection CSS import path
All symbols css/icon-packs/iso7010/icon-pack-iso7010.css
Emergency only css/icon-packs/iso7010/icon-pack-iso7010-emergency.css
Fire only css/icon-packs/iso7010/icon-pack-iso7010-fire.css
Mandatory only css/icon-packs/iso7010/icon-pack-iso7010-mandatory.css
Prohibition only css/icon-packs/iso7010/icon-pack-iso7010-prohibition.css
Warning only css/icon-packs/iso7010/icon-pack-iso7010-warning.css

These styles are not imported by default in the main project template and must be specifically added to your project copy.
